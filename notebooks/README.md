# Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and modeling.

## Notebook Structure

- `01_data_exploration.ipynb` - Initial data exploration and understanding
- `02_pattern_analysis.ipynb` - Pattern and relationship analysis
- `03_event_impact_modeling.ipynb` - Event impact estimation
- `04_forecasting_models.ipynb` - Time series forecasting models
- `05_visualization_dashboard.ipynb` - Dashboard development

## Usage

Each notebook is self-contained and can be run independently. Make sure to install dependencies first:

```bash
pip install -r requirements.txt
jupyter notebook
```
